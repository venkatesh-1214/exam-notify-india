# ExamNotify India

A government exam notification platform built using React.js and Firebase.

## 🚀 Features
- Latest government job notifications
- Search functionality
- Save jobs feature
- Real-time database

## 🛠️ Tech Stack
- React.js
- Firebase
- Firestore

# ExamNotify India 🇮🇳

A full-stack web application that provides real-time government exam notifications.

## 🚀 Features
- 🔍 Search government jobs
- 📄 Detailed job information
- ❤️ Save jobs for later
- 🔄 Real-time updates using Firebase

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Firebase
- Database: Firestore
- Deployment: Netlify
